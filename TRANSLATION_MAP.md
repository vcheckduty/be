# Backend Messages Translation Map

## Common Terms
- "Authorization token is required" → "Yêu cầu token xác thực"
- "Invalid or expired token" → "Token không hợp lệ hoặc đã hết hạn"
- "Internal server error" → "Lỗi máy chủ nội bộ"
- "Access denied" → "Truy cập bị từ chối"
- "User not found" → "Không tìm thấy người dùng"
- "Validation error" → "Lỗi xác thực"

## Check-in/Check-out → Điểm danh
- "check-in" → "điểm danh vào"
- "check-out" → "điểm danh ra"
- "checkin" → "điểm danh vào"
- "checkout" → "điểm danh ra"
- "Check-in" → "Điểm danh vào"
- "Check-out" → "Điểm danh ra"
- "attendance" → "chấm công"

## Roles & Permissions
- "Admin privileges required" → "Yêu cầu quyền Admin"
- "Only supervisors" → "Chỉ người giám sát"
- "Only officers" → "Chỉ nhân viên"
- "supervisor" → "người giám sát"
- "office" → "trụ sở"
- "approval" → "phê duyệt"
